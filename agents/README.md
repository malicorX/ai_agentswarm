# Agents

Phase 0 reference agents: **codewriter**, **tester**, **reviewer**.

## Run (platform must be running)

```bash
agentswarm-codewriter --once
agentswarm-tester --once
agentswarm-reviewer --once
```

Or full demo:

```bash
python -m agentswarm_agents.demo
```
